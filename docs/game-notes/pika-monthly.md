# 皮卡月刊（角色时装）

## 概念

皮卡月刊是洛克王国世界的角色时装系统，每期时装会绑定多个精灵，并有特殊的出场动画。

- **绑定精灵**：每期时装关联 3-4 个精灵，每个精灵有一男一女两个洛克时装
- **出场动画**：穿戴时装时有特殊精灵出场效果
- **概念图**：每期月刊有 2 张概念图（男洛克全时装展示 + 女洛克全时装展示）

## 时装分类

| 类型 | 说明 |
|------|------|
| 洛克男时装 | 某精灵的男性角色时装图片 |
| 洛克女时装 | 某精灵的女性角色时装图片 |
| 男概念图 | 该期所有男洛克时装的合照展示 |
| 女概念图 | 该期所有女洛克时装的合照展示 |

## 期数与周期

- **期数标识**：`YYYYMM`（年月），如 `202605` 表示 2026 年 5 月
- **每期持续时间**：约 2 周
- **精灵数量**：每期 3-4 只精灵

## 售卖与返厂机制

```
上架 ──(2周)──→ 下架 ──(冷却42天)──→ 进入「随机服装商店」
```

- **上架期间**：可直接购买当期月刊时装
- **下架后**：无法直接获取
- **冷却 42 天后**：进入游戏内「随机服装商店」
- **获取方式**：使用「许愿星」在随机商店抽取

## 许愿星获取

```
日常行为（捕捉等）→ 星光（每日有上限）→ 星光收集 → 许愿星
```

- **星光**：通过捕捉精灵等日常行为获取，每日获取存在上限
- **星光收集**：累积星光可兑换为许愿星
- **许愿星**：在随机服装商店中使用，获取已返厂的历史月刊时装

## 与活动的关联

每期皮卡月刊持续期间，会同步开启以下常驻课题活动：

| 活动 | 关联 |
|------|------|
| **命定花种** | 与当期月刊精灵一致（3-4 只），固定性格+三级亲密度 |
| **皮卡摄影委托** | 与当期月刊精灵一致，给精灵拍照赠送家园壁画 |

三者周期同步：月刊更新 → 命定花种更新 → 皮卡摄影委托更新

## 数据存储

- 主表：`pika_monthlies`（期数、名称、概念图男/女、上架/下架日期、排序）
- 关联表：`pika_monthly_pets`（月刊-精灵多对多，每个精灵存洛克男/洛克女时装图片）
- 主键：`pika_monthlies.id` 自增
- 关联：`pika_monthly_pets.pet_uid` → 精灵表

## 管理端操作流程

1. 创建新一期月刊（填写期数、名称、上架/下架日期）
2. 添加 3-4 只精灵（通过 PetPicker 选择）
3. 上传男/女概念图
4. 上传每只精灵的男/女洛克时装图片
5. 同步创建对应的「命定花种」和「皮卡摄影委托」活动
