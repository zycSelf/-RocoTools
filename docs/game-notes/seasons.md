# 赛季系统

## 赛季周期

- 游戏按赛季更新（S1、S2...）
- 每个赛季有独立的限定内容

## 通行证精灵

- 每赛季推出通行证，含 **2 只**通行证精灵可选
- 获取方式：
  - 自己拥有的果实
  - 去拥有果实的好友世界，通过重复捕捉概率获取

## 传说精灵

- 每赛季有 **1 只**主推的传说精灵
- 当赛季主要获取途径，不排除后续赛季返厂

## 赛季限定精灵

- 每赛季有 **8 只**仅当赛季可捕捉的精灵
- 这 8 只拥有异色版本
- 赛季结束后，异色版本仅可通过孵蛋获取

## 赛季异色精灵

- 每赛季有 **8 只**日常可捕捉的精灵，在当赛季可于野外获取其异色版本
- 赛季结束后，异色版本仅可通过孵蛋获取
- 精灵本体不受影响，随时可捕捉

## 赛季切换规则

- 赛季结束后，上述 16 只精灵（8 赛季限定 + 8 赛季异色）的异色版本：
  - 野外不再产出
  - 仅可通过**孵蛋**获取

## 数据结构（管理端配置）

| 字段 | 说明 |
|------|------|
| season_id | 赛季标识（如 S1、S2） |
| pass_pets | 通行证精灵（2只 uid） |
| legend_pet | 传说精灵（1只 uid） |
| season_pets | 赛季限定精灵（8只 uid，拥有异色） |
| shiny_pets | 赛季异色精灵（8只 uid，当赛季野外可获取异色） |
| announcement_url | 赛季更新公告链接（官方公告地址） |
| announcement_text | 公告横幅文案（首页展示） |
| announcement_content | 公告正文（Markdown格式，弹窗展示） |
