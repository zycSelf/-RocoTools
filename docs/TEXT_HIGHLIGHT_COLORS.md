## 四、属性系文本（动态匹配）

文本中出现 **"属性名+系"** 格式（如"火系"、"水系"、"恶系"）时，自动使用对应属性颜色高亮。

严格匹配格式：仅匹配 `{属性名}系`，不匹配其他组合。

| 文本 | 色号 |
|------|------|
| 普通系 | `#3F89B4` |
| 草系 | `#4EBC73` |
| 火系 | `#DB5525` |

## 五、匹配规则

1. **长词优先**：所有关键词按长度降序排序后进行正则匹配，确保"应对状态"优先于"应对"
2. **不重叠**：已匹配的文本不会被再次匹配
3. **加粗显示**：匹配的文本使用 `font-weight: 600` 加粗
4. **纯前端渲染**：通过 `v-html` 注入带颜色的 `<span>` 标签
